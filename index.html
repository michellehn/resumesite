Amply

www.amplyapp.com

Amply is a tool for creators, designed to amplify their presence in the blogosphere. The app analyzes tags or keywords that a user has chosen for original content, predicting how visible it will be online. In order to help users reach the largest possible audience, Amply suggests alternate terms to increase the power of their post.

I work on visual and web design for Amply using HTML, CSS, jQuery, Adobe Illustrator, and Adobe Photoshop.

  • Date: Fall 2014 - Present

TeaTime

Website

TeaTime is bubble tea and frozen yogurt store located in Ithaca, NY. Prior to this project, TeaTime did not have a website. A team of three other students and I worked over the course of a few months to design this website for the store, incorporating key components like business information, promotions, and a catering request form.
Languages used: HTML, CSS, PHP, jQuery

  • Role: Creator
  • Date: Winter 2014

Ithaca Apple Harvest Festival

Website

Every year, the town of Ithaca hosts the Apple Harvest Festival, a three day street festival devoted to apples, good food, and family fun. The original site for this event is difficult to navigate. So, the goal for this project was to create a site that is easy to use and understand, focusing on families as the target audience.
Languages used: HTML, CSS, jQuery

  • Role: Creator
  • Date: October 2014

K-Means Clustering

This algorithm clusters a set of data points into a set number k cluster groups. K centroids are picked, and each data point is partitioned to the nearest centroid. At each "step", the centroids are recalculated, and the data is repartitioned. This is continued until convergence is reached (when "Finished" is "True").

  • Role: Creator
  • Date: November 2014
